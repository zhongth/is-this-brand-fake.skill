# PDF 报告模板

品牌验真 PDF 报告的结构和格式说明。
生成报告时传递给 `pdf` skill 使用。

---

## 报告结构

### 第 1 页：封面 / 摘要

**标题：** 品牌验真报告

**字段：**
- **品牌：** {brand_name_en}（{brand_name_cn}）
- **声称产地：** {country}
- **调查日期：** {date}
- **判定结果：** {verdict} — 可信度 {confidence}%

**判定结果配色：**
- 正品品牌 → 绿色 (#22c55e)
- 大概率正品 → 浅绿 (#86efac)
- 待定 → 黄色 (#eab308)
- 大概率假洋牌 → 橙色 (#f97316)
- 假洋牌 → 红色 (#ef4444)

### 第 2 页：调查摘要

2-3 段文字概述：
- 调查了什么品牌、为什么调查
- 9 个步骤的关键发现
- 最具决定性的证据（最强的正面或负面信号）

### 第 3 页：证据总览表

| 调查步骤 | 信号评级 | 得分 | 关键发现 |
|---------|---------|------|---------|
| 政府与监管数据库 | {rating} | {score} | {一句话概括} |
| 商标时间线 | {rating} | {score} | {一句话概括} |
| 域名与网站 | {rating} | {score} | {一句话概括} |
| 零售渠道 | {rating} | {score} | {一句话概括} |
| 社交与职业足迹 | {rating} | {score} | {一句话概括} |
| 社区与媒体 | {rating} | {score} | {一句话概括} |
| SEO 内容质量 | {rating} | {score} | {一句话概括} |
| 中国实体交叉验证 | {rating} | {score} | {一句话概括} |
| **合计** | | **{total_score}** | **{verdict}** |

### 第 4-12 页：详细调查结果

每个调查步骤一个章节，每节包含：

**标题：** 步骤名称 + 信号评级标签

**检查了什么：**
- 访问的数据库/网站列表及 URL

**发现了什么：**
- 详细的调查发现描述
- 网页相关引文或数据
- 关键观察

**证据：**
- 来源 URL（可点击链接）
- 具体数据（注册日期、备案编号等）

**评级依据：**
- 为什么给出该评级（2-3 句话将证据与评级标准关联）

### 第 13 页：危险信号汇总

仅在存在"危险信号"评级时包含。

**已识别的危险信号：**
- 编号列表，每条标注对应步骤
- 示例："1.（监管）注册地址墨尔本 123 Example St 经 Google Maps 街景确认为汽修店"

### 第 14 页：调查方法

**本报告的生成方式：**

本报告由 is-real-brand 品牌验真工具生成，通过 9 个步骤系统调查品牌真实性。每个步骤搜索公开可用的数据库和网站，评估该品牌是否为真正的海外品牌，还是假洋牌。

**调查步骤：**
1. 品牌识别与准备
2. 政府与监管数据库核查（权重 20%）
3. 商标时间线分析（权重 15%）
4. 域名与网站分析（权重 10%）
5. 零售渠道存在度（权重 15%）
6. 社交与职业足迹（权重 10%）
7. 社区与媒体存在度（权重 10%）
8. SEO 内容质量分析（权重 10%）
9. 中国实体交叉验证（权重 10%）

**评分方式：** 每个步骤产生一个信号评级（已验证 / 可疑 / 未找到 / 危险信号），对应一个数字分数。总分范围为 -100 到 +100，决定最终判定结果。

### 第 15 页：免责声明

**免责声明：**

本报告通过自动化网络调研和 AI 分析公开信息生成，仅供参考，不构成法律建议、监管认定或品牌真实性的最终判断。

局限性：
- 结果依赖于调查时可获取的公开信息
- 某些数据库在调查期间可能暂时不可用
- 新品牌或小型正规品牌可能因在线信息有限而得分较低
- 本工具无法验证实物产品质量或成分真实性

如需权威验证，请咨询品牌声称来源国的相关监管机构。

## 格式要求

- 使用简洁专业的排版
- 信号评级标签：使用彩色标签（绿/黄/灰/红）
- 包含调查日期和唯一报告编号
- 所有 URL 使用完整链接，不缩短
- 结构化数据使用表格
- 每个步骤的详细章节控制在 1-2 页以内
- **全文使用中文**，品牌英文名、URL 等保留原文
